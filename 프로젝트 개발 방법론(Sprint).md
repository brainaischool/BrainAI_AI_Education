## 💬 Sprint 용어 정리

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Sprint 관련 용어 사전
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Sprint
   = 짧은 기간 (1-2주)
   예: "Sprint 2가 시작됐어요!"


2. Sprint Goal
   = 이번 Sprint의 목표
   예: "데이터 300장 수집"


3. Sprint Planning
   = Sprint 시작 회의
   예: "이번 주 뭐 할까?"


4. Daily Standup
   = 매일 아침 짧은 회의
   예: "어제/오늘/문제" 공유


5. Sprint Review
   = Sprint 끝 시연
   예: "300장 모았어요! (보여주기)"


6. Retrospective
   = 회고, 돌아보기
   예: "뭐가 좋았고, 뭘 개선할까?"


7. Sprint Backlog
   = Sprint 동안 할 일 목록
   예: 
   □ 카메라 연결
   □ 데이터 수집
   □ 데이터 확인


8. Burndown Chart
   = 진행률 그래프
   예: 
   Day 1: 남은 일 10개
   Day 3: 남은 일 5개
   Day 5: 남은 일 0개! ✅
```
