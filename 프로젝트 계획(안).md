#### **4.2 우리 프로젝트 네이밍 규칙**

```
📸 이미지 파일 이름
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

형식: 0001.jpg, 0002.jpg, ...

규칙:
✓ 4자리 숫자 (0001~9999)
✓ 순서대로 증가
✓ 앞에 0을 채워서 정렬 편하게

예시:
0001.jpg  ← 첫 번째 사진
0002.jpg  ← 두 번째 사진
...
0350.jpg  ← 350번째 사진

📝 어노테이션 파일 이름
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

형식: 0001.json, 0002.json, ...

```
규칙:
✓ 이미지 파일과 똑같은 번호
✓ 1:1 짝꿍 관계
✓ JSON 형식

예시:
0001.jpg ↔ 0001.json  (짝꿍!)
0002.jpg ↔ 0002.json  (짝꿍!)

0001.json 내용:
{
    "image_id": "0001",
    "steering": 0.5,      ← 핸들 각도
    "speed": 600,         ← 속도
    "timestamp": "..."    ← 촬영 시각
}


─────────────────────────────────────

🐍 Python 파일 이름
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

형식: 소문자_언더스코어_사용.py

규칙:
✓ 모두 소문자
✓ 띄어쓰기 대신 _ (언더스코어)
✓ 기능이 명확하게 드러나게

예시:
✓ car_controller.py      ← 차 제어
✓ data_collector.py      ← 데이터 수집
✓ model_train.py         ← 모델 훈련

❌ CarController.py       (대문자 X)
❌ datacollector.py       (띄어쓰기 없으면 읽기 힘듦)
❌ train.py               (너무 애매함)


─────────────────────────────────────

📓 노트북 파일 이름
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

형식: Step번호_한글설명.ipynb

규칙:
✓ Step 번호로 순서 표시
✓ 한글로 내용 설명
✓ 띄어쓰기 대신 _ 사용

예시:
✓ Step1_데이터_획득.ipynb
✓ Step2_데이터_병합.ipynb
✓ Step7_도로_주행.ipynb

❌ step1.ipynb            (무슨 내용인지 모름)
❌ 데이터 획득.ipynb      (순서를 모름)


─────────────────────────────────────

📁 폴더 이름
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

형식: 소문자 또는 snake_case

규칙:
✓ 짧고 명확하게
✓ 영어 사용 권장
✓ 복수형 사용 (여러 파일 담으니까)

예시:
✓ configs/               ← 설정들
✓ utils/                 ← 도구들

✓ data/raw/              ← 원본 데이터들

❌ Config/               (대문자 X)
❌ my-utils/             (하이픈 X)

#### **7.1 자주 만나는 문제들**

```
🔧 문제 1: 마이크로비트가 안 보여요!
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

증상:
"Micro:bit을 찾을 수 없습니다" 메시지

해결 순서:
1단계: USB 케이블 다시 꽂기
  └→ 안 되면 2단계

2단계: 장치 관리자 확인
  └→ COM 포트 보이나요?
  └→ 안 보이면 3단계

3단계: 드라이버 재설치
  └→ 마이크로비트 공식 사이트

4단계: 다른 USB 포트 시도

5단계: 선생님께 도움 요청!


🔧 문제 2: PS4 컨트롤러 연결 안 돼요!
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

증상:
"PS4 컨트롤러를 찾을 수 없습니다"

해결 순서:
1단계: 페어링 다시 하기
  SHARE + PS 버튼 3초

2단계: Bluetooth 설정 확인
  └→ "Wireless Controller" 보이나요?

3단계: 다른 기기와 연결 해제
  └→ 스마트폰이랑 연결되어 있나요?

4단계: 컨트롤러 충전 확인
  └→ 불빛 켜지나요?


🔧 문제 3: 카메라 화면이 안 나와요!
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

증상:
검은 화면만 보여요

해결 순서:
1단계: 카메라 번호 확인
  config: camera: source: 0
  └→ 1로 바꿔보기

2단계: 다른 프로그램이 카메라 사용 중?
  └→ Zoom, Teams 끄기

3단계: 카메라 권한 확인
  └→ Windows 설정 → 개인정보


🔧 문제 4: 훈련이 너무 느려요!
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

증상:
1 epoch에 10분 이상...

해결 방법:
1. epochs 줄이기 (50 → 30)
2. batch_size 늘리기 (32 → 64)
3. 데이터 수 줄여보기 (테스트용)
4. Google Colab 사용하기 (GPU 무료!)


🔧 문제 5: 자율주행이 계속 실패해요!
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

증상:
차선 이탈, 충돌, 정지

점검 사항:
□ 조명이 훈련 때와 비슷한가요?
□ 트랙이 훈련 때와 같나요?
□ 속도가 너무 빠른가요?
  deployment.yaml → base_speed 줄이기

□ 데이터가 충분한가요?
  최소 300장 필요!

□ 훈련이 잘 됐나요?
  val_loss 확인!
```
